Este plugin cria links para produtos no Zura!

== Description ==

Este plugin tem a finalidade de criar links dinâmicos para produtos do Zura! nos blogs dos seus parceiros. O plugin fara:

   1. Verificara o conteudo das materias em busca de algumas palavras pre-selecionados pelo Zura!
   2. Criara links para os produtos passando o PR e o site do afiliado.
   3. Os links poderao ser abertos na mesma pagina ou em uma nova janela
   

== Installation ==

1. Faca upload dos arquivos para a pasta `/wp-content/plugins/`  (mantenha a pasta original do plugin)
2. Ative o plugin na interface de 'Plugins' do WordPress
3. Na aba de opcoes (Zura HotLinks) voce devera configurar seu plugin.
3.1. Cadastre seu Identificador de parceiro(PR)
3.2. Caso possua mais de um site, cadastre-o tambem (site)
3.3. Selecione o tipo de codificacao utilizado em seu blog (ISO-8859-1, UTF-8, etc.) O plugin foi pre-configurado para utilizar UTF-8
3.4. Por ultimo, marque as opcoes que melhor lhe servirem (links abrindo em nova pagina e numero maximo de links por post)
